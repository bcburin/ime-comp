
# Gramática Ambígua

Uma [[LFA - Gramática Livre de Contexto|gramática livre de contexto]] é **ambígua** se existe pelo menos uma [[LFA - String|sentença]] na linguagem por ela gerada que pode ser representada por mais de uma [[LFA - Árvore de Derivação|árvore de derivação]], ou seja, que possui mais de uma [[LFA - Derivação|derivação]] possível.
