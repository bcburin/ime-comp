# Sequência de Formação

---

### Definição

Uma sequência $\phi_0, \dots, \phi_n$ é de **sequência de formação** de $\phi$ se e só se $\phi_n = \phi$ e, para todo $i \leq n$, 

- $\phi_i$ é atômica; ou 
- $\phi_i = (\phi_j \;\square\; \phi_k)$, para $j, k < i$; ou
- $\phi_i = (\neg \phi_j)$, para $j < i$.
