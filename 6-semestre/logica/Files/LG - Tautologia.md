# Tautologia

---

### Definição

Uma proposição $\phi$ é uma tautologia se, para toda valoração $v$, $v(\phi)=1$.

**Notação:** $\models \phi$

**Exemplo:** $\models \Big( \big(\,(\phi \wedge \psi) \rightarrow \gamma\,\big) \leftrightarrow \big(\,\phi \rightarrow ( \psi \rightarrow \gamma) \,\big) \Big)$
