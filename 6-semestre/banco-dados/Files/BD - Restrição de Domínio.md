
# Restrição de Domínio

---

- Para um [[BD - Esquema de uma Relação|esquema]] $R$, o valor de cada [[BD - Atributo|atributo]] $A_j$ deve pertencer ao [[BD - Domínio|domínio]] $\mathrm{dom}(A_j)$
- Geralmente, o domínio é especificado através de tipos primitivos, tais como **int**, **float**, **char**, **date**, **time**, etc.
