# Número de Weber

---

### Definição

Razão entre força de inérica e tensão superficial

$$
\boxed{
We = \frac{\rho V^2 L}{\Gamma}
}
$$

Em que

- $\rho$ => [[FT - Massa específica|massa específica]]
- $V$ => velocidade de escoamento
- $L$ => dimensão de referência
- $\Gamma$ =>  tensão superficial
