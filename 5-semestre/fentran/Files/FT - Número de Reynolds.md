# Número de Reynolds

---

### Definição

Razão entre força de inérica e força viscosa

$$
\boxed{
Re = \frac{\rho V L}{\mu}
}
$$

Em que

- $\rho$ => [[FT - Massa específica|massa específica]]
- $V$ => velocidade de escoamento
- $L$ => dimensão de referência
- $\mu$ => [[FT - Viscosidade#Viscosidade Dinâmica|viscosidade dinâmica]]
